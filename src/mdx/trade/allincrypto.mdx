## 他“做起来”的几个关键要素
下面是熬鹰在公开分享中提及的、他认为自己从亏损中走出来、并取得大规模收益的关键要素。

1. 学习方式：更多实战＋借鉴而不是死学指标
  - 以前他也学过很多经典技术指标／策略，比如双均线、EMA、波浪理论、道氏理论、海龟法则等。 
  - 但他后面总结：这些指标只能帮助“点位开得稍好一点”，但不能决定你能否做出大钱。指标胜率随行情变化波动很大。 
  - 他更多是“观察真实交易员的实盘”＋“请教他们的单子逻辑”＋不断实战摸索。 

2. 心态与资金／仓位管理非常重要
  - 他强调：交易不是“看指标赢”的游戏，而是“认知博弈”与“心理成熟度”的问题。 
  - 在资金量很小、负债累累时，他体验到“亏得还不痛”“心理没整理好”是大问题。 
  - 关于合约操作，他建议：杠杆别太高，尤其是初期；资金滚起来、认知提升比一味冲高杠杆更重要。 

3. 交易逻辑／风格：事件驱动 ＋ 结构判断
  - 他现在的交易风格倾向于“事件驱动型”＋“看结构”而不是仅看指标。即：先判断某个宏观／链上／市场情绪事件能否触发行情，再看价格结构是否配合。 
  - <LinkRender alt='传奇交易员“熬鹰”' path='https://www.binance.com/zh-CN/square/post/27319311984241' />
  - 他会看一小时级别、20-50根K线的走势段落，判断是否处于震荡末期或突破点。 

4. 从小市值／山寨币＋热点行情起步，逐步转向较大币种或稳健风格
  - 起初他、像很多小资金交易者一样，参与主流热点／小市值山寨／热门模因币。 
  - <LinkRender alt='传奇交易员“熬鹰”' path='https://blockweeks.com/kol/130642' />
  - 随着资金规模变大，他意识「资金量上来了就不能再用小资金的打法」，需要调整交易周期、仓位控制与品种选择。 

## “做起来”——他所谓的转折点
  - 他曾说，在2024 年他抓住了三波大行情（包括 AI 币、模因币、铭文coin 第二春）——这些大趋势给了他资金规模快速成长的机会。 
  - 他打通了“20万人民币”这个资金瓶颈（他称“20万魔咒”），从几十万长期停滞，到踏出、负债还清、资金开始指数级增长。 

熬鹰“加密做起来”并不是靠一个神秘指标，而是靠：从小资金实战、经历亏损、提升认知、用事件＋结构逻辑判断行情、控制风险／仓位、在热点行情中抓机会。渐渐规模起来，再进入带单与社群层面

## 一、早期：靠「反人性」打法熬出经验（2021–2022）

1. 初期他一直亏 —— 因为跟大多数人一样「追涨杀跌」

他一开始也是学各种指标，看到涨就上、跌就割，爆仓十几次。
他说：「我第一次赚钱，是在我彻底放弃预测、改用概率思维之后。」

2. 他意识到：“行情不是分析出来的，是等出来的。”
这时候他开始：
  - 只做自己看得懂的结构（比如趋势延续或趋势反转的形态）
  - 等突破确认再进，而不是猜底或猜顶
  - 一旦错了就止损（很快走）

这让他开始减少亏损次数，哪怕还没持续盈利，但亏得更慢。

## 二、中期：靠「事件驱动 + 结构单」突破（2023年）
他真正“做起来”，是在这一阶段开始的。
核心是：他发现纯技术无效，但「事件 + 结构」能形成高胜率机会。

1. 「事件驱动」指什么？
    - 他会提前布局能引爆板块的事件，比如：
    - 以太坊升级（上海升级那波他做了 ETH）
    - 模因币狂潮（Pepe、Bonk 等）
    - 铭文coin二次爆发（他参与了BTC铭文、Doginals那波）

他称之为：“事件确认 + 结构共振 = 黄金入场点”。

2. 「结构单」怎么打？ 结构单 = 观察K线结构、趋势线、箱体突破点：
  - 确认一小时级别突破（价格突破区间并量能放大）
  - 在回踩确认时上车
  - 止损设在结构下沿，止盈按1:3或以上的盈亏比
  - 有时候他会用网格或分批加仓来「吃掉一整段波段」

举例：2023年他在 Pepe 上赚到人生第一桶金。
他提前看到社群热度异常＋推特搜索暴涨→进场早，持有到行情情绪顶点才减仓，翻了近20倍。

## 三、转折点：从“短线爆发”到“稳健复利”（2024年）
1. “20万魔咒”阶段突破
他在直播里说过：“我卡在20万人民币资金规模快两年”。
后来是通过 事件中长线 + 小杠杆复利 才突破。

操作逻辑：
- 用小仓位试单（1x–3x杠杆），若方向对，加仓；
- 若方向错，亏损控制在 2%–3%；
- 每次行情抓住波段，就吃 30%–50%；
- 资金不断放大后，不再追求爆倍，而是稳复利。

他自己说那时候“开始能一个月做 +30%，连做几个月”。

2. 控仓法则
  - 单次仓位控制在账户 20%–30%；
  - 止损绝不超过本金 5%；
  - 情绪交易一律禁止（比如刚爆仓后不准开单）。

这些让他从“偶尔爆赚爆亏”转成“持续赚钱”。


## 四、他靠交易赚大钱的核心模型（可总结成四步）
| 阶段         | 动作      | 说明                     |
| ---------- | ------- | ---------------------- |
| **1. 潜伏期** | 找事件、找结构 | 比如 AI 板块要起，他先看结构形态是否筑底 |
| **2. 启动期** | 小仓试探    | 小仓,小于20% 先测方向，亏也不心疼     |

他常说一句话：“一轮行情，只要你能吃到中间那一段，就赢了。”

总结成一句话：

熬鹰是这样靠交易做起来的：

从盲目指标 → 概率思维 → 事件驱动 + 结构单 → 小杠杆复利 + 严控风险 → 资金滚大后再品牌化。

他不是靠“预测行情”赚的，是靠等待+纪律+理解市场节奏。

# 怎么做起来
真相是：他起家靠的是 BTC/ETH 等主流币合约波段交易，
但后来才把“资金放大”的关键一击打在「山寨币与热点事件」上。

## 阶段一：靠 BTC 合约磨出来的功底（2021–2022）
熬鹰最早接触的是「币本位合约」，特别是 BTC 永续合约。
他在这阶段不是赚钱阶段，而是“练出系统”的阶段。

重点在「方向判断 + 仓位节奏」：

* 他当时每天做短线，频繁爆仓。后来开始改成：
    - 只做趋势确认后的顺势单（不猜底）
    - 固定止损、固定加仓节奏（比如 2x 杠杆 + 分批进）

* 他发现，BTC 合约是最能训练「方向感 + 纪律」的。
    - 因为它波动大，但趋势清晰（不像小币容易被庄家洗）。

熬鹰原话（访谈中提到）：
“我在 BTC 上亏的钱，就是后来赚的钱的学费。
一开始我把它当赌场，后来才明白——BTC 是最纯粹的趋势资产。”

## 阶段二：靠「事件型山寨行情」实现爆发（2023）, 这时他才赚到第一桶金。
当他 BTC 合约技术已经稳定后，他开始利用「热点叙事」做山寨币波段。

做法不是盲炒，而是：
1. 先看 BTC 结构：确认稳住或上行（大盘安全）
2. 再找叙事：比如 Pepe、铭文、AI、RWA
3. 用事件 + 结构共振逻辑（真突破）
4. 轻杠杆（2–3x）吃波段

Pepe 那波（2023年4月）是他的转折点:
1. 他提早看到社群疯传、X 搜索量暴涨
2. Pepe 刚在币安上线时突破关键区间
3. 他在早期进场、一路拿到主升浪
4. 翻了十几倍（这波直接让他账户翻数十倍）

熬鹰说： “靠 BTC 练手，靠山寨起飞。”

## 阶段三：BTC 合约稳 + 山寨波段放大（2024–2025）
他现在的体系已经是「双轨模式」：

| 类型             | 作用          | 风格                      |
| -------------- | ----------- | ----------------------- |
| **BTC/ETH 合约** | 主仓位 / 稳定复利  | 做趋势波段，1x–3x杠杆，月化 10–30% |
| **山寨币 / 事件币**  | 辅助仓位 / 放大利润 | 抓热点波段，短期高收益             |

他自己说：
“BTC 给我底气，山寨给我惊喜。
没 BTC 的结构确认，我不会碰小币。”

## 一句话总结：

熬鹰不是靠乱炒山寨币起家，
而是靠 BTC 合约打基础 → 学会结构判断 → 在山寨行情中精准出击 做起来的。

他真正赚钱的能力来自：
1. 对 BTC 结构的极强判断力
2. 把握热点周期的节奏感
3. 严格的仓位管理与心态控制。
