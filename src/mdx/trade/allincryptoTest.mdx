## 一、核心思路（照熬鹰的逻辑，适合你当前阶段）
目标不是暴富，而是活下来 + 稳定复利。
熬鹰自己前期爆了几十次仓，他后来说过：“第一阶段的胜利，不是赚，而是亏得慢。”

你现在要进入的是熬鹰早期「BTC磨功底」阶段。

1. 主目标：建立稳定可重复的交易体系
2. 核心：方向判断、仓位节奏、止损纪律
3. 禁忌：高杠杆赌方向、连亏后复仇单、频繁进出

## 二、具体执行框架（按阶段分）
## 阶段1：建立结构感（1–2个月）

每次出手都有依据，不预测，只交易「确认」后的走势。

操作策略
1. 仅做 BTC/USDT 永续合约
2. 时间周期：1小时K线
3. 工具：支撑压力线 + EMA20/50 结构参考
4. 模式：等待震荡区间突破 → 回踩确认进场

举例（结构单逻辑）：
1. BTC 价格横盘 6–12小时 → 形成区间
2. 突破区间上沿（或下沿）+ 放量 → 结构确认
3. 回踩时小仓入场（1–2x杠杆）
4. 止损：设在结构下沿（亏损≤账户2%）
5. 止盈：盈亏比 1:3 起步

仓位建议：
1. 总资金4000U
2. 单次风险 ≤ 2%（即最大亏损80U）
3. 若止损距离为2%，则仓位 = 4000 × 0.02 ÷ 0.02 = 4000U，杠杆1x
4. 若止损距离为1%，杠杆可提到2x

结论：1–3x杠杆，小仓＋止损清晰。


## 阶段2：建立节奏感（2–3个月）
训练「等」的能力，不乱出手，只做“行情段落”。

训练目标：
1. 控制“手痒”
2. 不在模糊区间开单（横盘不碰）
3. 找到自己最容易看懂的结构形态

## 阶段3：加仓与复利（第4个月起）
有了稳定正收益后，再轻杠杆复利。

规则：
1. 当账户从 4000 → 6000U 后
  - 每次风险仍占账户2%，但仓位自然增大

2. 目标月化收益 15–25%

3. 加仓法：
  - 小仓试探（确认后再加）
  - 每次加仓后设好保护止损（保本止损）

举例：
1. 初仓 1000U（1x） → 行情确认后加 500U（共1.5x）
2. 若反向1%，全止损；若顺势+3%，止损抬至保本
3. 拉开后逐步减仓，不追顶
