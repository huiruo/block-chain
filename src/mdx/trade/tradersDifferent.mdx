能长期成功做交易的人，确实常常会被外界觉得“有点疯”，但那种“疯”其实更多是一种反常识的理性与纪律。

成功的交易者之所以像疯子，是因为他们在一个充满情绪的世界里保持了极端的理性。

### 一、从普通人的角度看，他们像“疯子”
- 大多数人讨厌亏损、追求确定性； 
  - 而成功的交易者愿意不断止损、接受不确定性。
  - 在旁人看来，这像是“故意去受苦”。

- 普通人会在行情剧烈时被情绪左右；
  - 成功的交易者却在恐慌中冷静加仓或反向操作。 看起来“逆天而行”，甚至“疯了”。
  - 普通人追求“正确”；
  - 成功的交易者知道多数时候自己是错的，只要控制风险、让正确的那部分赚回来就行。

###  二、从心理结构看，他们更像“理性的疯子”
这在一般社会行为标准下，确实显得“冷血”或“反人性”。他们的思维方式与常人不同：
- 他们能把金钱游戏当成概率游戏；

- 他们的自我情绪和决策系统是分离的；

- 他们在巨大波动中依然能睡得着；

- 他们不靠感觉，而靠系统与纪律。

### 三、真正的交易成功不靠“疯狂”，靠的是“自控”
所谓的“疯”，其实是一种高度自控+理性偏执的状态。
他们不是情绪上失控的疯子，而是能控制住疯狂冲动的人。

这才是区别：

| 类型   | 特征           | 结果   |
| ---- | ------------ | ---- |
| 情绪疯子 | 赌徒心态，冲动进出场   | 爆仓   |
| 理性疯子 | 严格执行系统、反人性操作 | 长期盈利 |
